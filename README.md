# Astraeus-Framework

## Introduction
Astraeus is an open-source RuneScape emulator built using Java NIO.

## Why use Astraeus?
Astraeus strives to become a high-performance server. The content of Astraeus will be using plugins (I have not decided on the scripting language yet)
The source is very clean, and well documented.
